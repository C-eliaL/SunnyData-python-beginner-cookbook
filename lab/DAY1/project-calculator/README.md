## 🛠️ 開發指南（漸進式）

### 階段 A：完成輸入功能 (20 分鐘)
- **目標**：使用 `input()` 取得使用者輸入並正確轉換型態。
- **步驟**：
  1. 使用 `float(input("請輸入第一個數字: "))` 取得第一個數字
  2. 使用 `float(input("請輸入第二個數字: "))` 取得第二個數字
  3. 使用 `print()` 確認輸入是否正確
- **檢核點**：
  - [ ] 所有輸入正確取得
  - [ ] 數字輸入轉成浮點數（支援整數和小數）
  - [ ] 變數命名清楚（例如：`num1`, `num2`）
- **提示**：使用 `float()` 而非 `int()`，可同時支援整數和小數輸入。

### 階段 B：實作計算邏輯 (15 分鐘)
- **目標**：執行兩個數字的加法運算。
- **步驟**：
  1. 使用加法運算子 `+` 計算結果
  2. 將結果儲存在變數中（例如：`result`）
  3. 使用 `print()` 顯示計算結果
- **檢核點**：
  - [ ] 加法運算正確：`10 + 5 = 15`
  - [ ] 小數運算正確：`3.5 + 2.5 = 6.0`
  - [ ] 負數運算正確：`-5 + 3 = -2`
- **範例輸出**：
```
請輸入第一個數字: 10
請輸入第二個數字: 5
計算結果：15.0
```

### 階段 C：格式化輸出 (20 分鐘)
- **目標**：使用 f-string 以易讀格式顯示計算結果。
- **步驟**：
  1. 加入歡迎訊息
  2. 使用 f-string 格式化輸出
  3. 格式：`數字1 + 數字2 = 結果`
  4. 加入結束訊息
- **檢核點**：
  - [ ] 使用 f-string 格式化
  - [ ] 輸出格式清楚易讀
  - [ ] 包含運算符號和等號
- **範例輸出**：
```
=== 歡迎使用簡易加法計算機 ===
請輸入第一個數字: 10
請輸入第二個數字: 5

計算結果：10.0 + 5.0 = 15.0
感謝使用計算機！
```

### 階段 D：進階思考 (可選 10 分鐘)
- **思考題**：如果要支援減法、乘法、除法，需要什麼新的知識？
- **預告**：在未來的章節（Ch04 條件判斷）中，我們會學習如何根據使用者的選擇執行不同的運算。
- **挑戰**：嘗試計算三個數字的總和（例如：10 + 5 + 3 = 18）

## ⚠️ 常見錯誤

1. **忘記轉型**：`input()` 回傳字串，未轉換成數值導致運算錯誤（例如：`"10" + "5" = "105"`）。
2. **使用 int() 而非 float()**：無法處理小數輸入，輸入 `3.5` 會產生錯誤。
3. **變數命名不清楚**：使用 `a`, `b`, `c` 等單字母變數，降低可讀性。
4. **未使用 f-string**：使用舊式字串格式化或字串連接，程式碼較不清晰。
5. **輸出格式混亂**：未加入適當的空行或分隔，輸出不易閱讀。
